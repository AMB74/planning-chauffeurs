# Planning Chauffeurs 🚗

Gestion du planning et des lettres de voiture pour les chauffeurs.

## 📋 Description

Ce projet permet de :
- Organiser et consulter le planning des chauffeurs
- Générer et gérer les lettres de voiture
- Centraliser les informations de disponibilité et d'affectations

## 🚀 Démarrage rapide

### Prérequis
- Node.js (version recommandée)
- Un navigateur moderne

### Installation

1. Clonez le repository :
```bash
git clone https://github.com/AMB74/planning-chauffeurs.git
cd planning-chauffeurs
```

2. Installez les dépendances :
```bash
npm install
```

3. Démarrez l'application :
```bash
npm start
```

## 📖 Utilisation

- Accédez à l'application via votre navigateur
- Consultez le planning des chauffeurs
- Créez ou mettez à jour les lettres de voiture selon vos besoins
